# tampermonkey

个人 Tampermonkey 脚本集合。

## 洛谷随机题

洛谷题库随机刷题辅助脚本，适用于 Tampermonkey / Violentmonkey。

### 简介

在洛谷题库页和题目详情页添加一个“洛谷随机题”浮动面板，用于按指定难度随机打开一道题目。脚本支持过滤已 AC 题、拉黑题目、缓存页数、快速刷新，并提供拖拽、缩放、最小化和位置记忆等功能，让随机刷题更顺手。

### 功能

* 按 7 档洛谷难度随机题目
* 可选择是否随机已 AC 题
* 支持拉黑当前题目
* 支持查看、移除、清空拉黑列表
* 随机后自动最小化，不遮挡题面
* 最小化后可点击当前难度继续随机
* 缓存每个难度的最大页数
* 使用“相邻页题号相同即越界”的方式探测最大页，避免洛谷超页显示最后一页导致误判
* 面板支持拖拽、缩放、字体调整
* 自动记住面板位置、大小和字体

### 使用

安装脚本后，打开洛谷任意页面：

```text
https://www.luogu.com.cn/*
```

右下角会出现“洛谷随机题”浮动面板。

选择难度后点击：

```text
随机一道题
```

即可随机打开对应难度的一道题。

在题目详情页中，可以点击：

```text
拉黑当前
```

将当前题目加入拉黑列表，之后随机时会自动跳过。

### 备注

脚本会读取洛谷题库列表页中的题号和通过状态，不会一次性爬取全部题目。页数缓存、拉黑列表、AC 开关和 UI 设置均保存在浏览器本地。
