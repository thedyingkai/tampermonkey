# tampermonkey
个人 Tampermonkey 脚本集合

## 洛谷随机题

洛谷题库随机刷题辅助脚本，适用于 Tampermonkey / Violentmonkey。

### 简介

在洛谷题库页和题目详情页右下角添加一个“洛谷随机题”浮动面板，用于按指定难度随机打开一道洛谷题目。

### 功能

* 按 7 档洛谷难度随机题目
* 可选择是否随机已 AC 题
* 支持拉黑当前题目
* 支持查看、移除、清空拉黑列表
* 缓存每个难度的最大页数
* 最大页探测使用“相邻页题号相同即越界”的方式，避免洛谷超页显示最后一页导致误判
* 面板支持拖拽、缩放、字体调整
* 自动记住面板位置、大小和字体

### 使用

打开以下页面后，右下角会出现脚本面板：

```text
https://www.luogu.com.cn/*
```

选择难度后点击“随机一道题”即可。

### 备注

脚本会读取洛谷题库列表页中的题号和通过状态，不会一次性爬取全部题目。页数、拉黑列表和 UI 设置均保存在浏览器本地。
